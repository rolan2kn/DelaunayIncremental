Proyecto Final: Implementación de una biblioteca de gestion de poliedros.
Autor: Rolando Kindelan Nuñez.

Para ejecutar el proyecto es necesario utilizar el CLion.

Se implementaron la mayoria de las clases:

incluyendo la clase poliedro, poligono, vector, boundingSphereOcTree, avl, entre otras clases y estructuras... 

Falto implementar el metodo para generar poliedro y el metodo para chequear si un poliedro es simple. 

Falto realizar pruebas exhaustivas... 

El codigo no tiene implementada interfaz grafica porque el objetivo era implementar la clase poliedro. 

Muchos algoritmos estan implementados con HalfEdge, en la clase poligono y poliedro.
